# @query-desk/assets

Shared assets package for the Query Desk workspace.

## Usage

### Images

```javascript
import image from "@query-desk/assets/images";
```

### SVG Icons

```javascript
import icon from "@query-desk/assets/svg";
```

## Directory Structure

- `images/` - Image assets (PNG, JPG, WEBP, etc.)
- `svg/` - SVG icons and graphics
